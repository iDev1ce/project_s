How your STIX bundle is generated
When you upload a report, the server will:
1. Receive and prepare the document for processing
2. Extract observables and threat intelligence entities from the content
3. Review and refine the extracted data to improve quality and remove unsupported items
4. Build relationships between the verified entities
5. Export the final result as a STIX 2.1 bundle in JSON format
6. Return the generated bundle for download
